# Zone Zero: 2D Survival

A top-down 2D survival sandbox game built with Python and Pygame.

## Created By
**Tejas R Gowda**
*with assistance from Antigravity (Google DeepMind)*

## Controls
- **W, A, S, D**: Move Player
- **E**: Pick Up Item
- **ESC**: Quit Game

## Gameplay
- **Survive**: Manage your Health, Hunger, and Thirst.
- **Scavenge**: Find Apples (Food), Water (Thirst), and Medkits (Health).
- **Evade**: Avoid Zombies. They will chase you if you get too close!

## Installation
1.  Install Python 3.x
2.  Install Pygame:
    ```bash
    pip install pygame
    ```
3.  Run the game:
    ```bash
    python main.py
    ```

## Features
- Infinite scrolling camera
- Procedural-style map with collisions
- Inventory system
- Basic Enemy AI
- Survival stats system
